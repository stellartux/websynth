Class: Note

Note(target, noteParams, oscParams)

Class representing a note composed of one or more oscillators.

Constructor

new Note(target, noteParams, oscParams)

Create a note
Parameters:
Name Type Description
target AudioContext | AudioNode output destination for audio
noteParams Object options that affect the note as a whole
Properties
Name Type Description
attack number attack time in seconds, defaults to 20ms
decay number decay time in seconds, defaults to 20ms
release number release time in seconds, defaults to 10ms
sustain number sustain level as proportion of peak level, 0 to 1 inclusive
triggerTime number time to schedule the note (see AudioContext.currentTime)
velocity number note velocity, 0 to 1 inclusive
oscParams Array.<Object> array of oscillator specific options
Properties
Name Type Description
detune number detune amount of the oscillator in cents
frequency number frequency of the oscillator in Hertz
gain number gain of the oscillator, 0 to 1 inclusive
type string waveform shape, options are "sine", "square", "sawtooth", "triangle"
